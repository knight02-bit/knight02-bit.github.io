    [linux虚拟机] 使用yum命令时，解析不了yum源，Cannot find a valid baseurl for repo base 7 x86 6

    起因:

    帮同学在centOS虚拟机中安装gcc

    yum -y install gcc gcc-c++ kernel-devel

    .png){.img-fluid tag=3}

    但未成功👇

    若是你也是这样,请千万先ping一下看看虚拟机是否能联网

    ping 114.114.114.114

    .png){.img-fluid tag=3}

    若是不能, 先查看网卡名

    ip addr

    然后

    vi /etc/sysconfig/network-scripts/ifcfg-网卡名字

如：vi /etc/sysconfig/network-scripts/ifcfg-ens33

    .png){.img-fluid tag=3}

    把ONBOOT=no 改为 ONBOOT=yes

    记得把当前虚拟机的网络适配器设为NAT模式

    然后重启网络(务必执行, 很多滥竽充数的教程不会提示,如果不重启依然ping不通(•́へ•́╬))

    service network restart

    .png){.img-fluid tag=3}

    此时重新执行你的yum命令,如

    yum -y install gcc gcc-c++ kernel-devel

    .png){.img-fluid tag=3}

    祝你好运
